# Forex Dashboard

React-based web dashboard for monitoring forex signals and managing the agent.

## Features (Planned)

- Real-time signal monitoring
- Historical signal analysis
- Configuration management
- User authentication
- Trading pair management

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The dashboard will be available at http://localhost:5173

## Build

```bash
npm run build
```

## Technologies

- React 18
- TypeScript
- Vite
- TanStack Query (React Query)
- React Router
- Axios
